## Минимальные рабочие примеры

### {Сценарий 1: типичный запрос}

```python
import os
import requests

API_KEY = os.environ["RAPIDAPI_KEY"]
HOST = "{host}.p.rapidapi.com"

headers = {
    "X-RapidAPI-Key": API_KEY,
    "X-RapidAPI-Host": HOST,
}

r = requests.get(
    f"https://{HOST}/{endpoint}",
    headers=headers,
    params={"param1": "value"},
    timeout=15,
)
r.raise_for_status()
print(r.json())
```

### {Сценарий 2: пагинация, если применимо}

```python
def paginate(path, params, max_pages=5):
    token = None
    for _ in range(max_pages):
        if token:
            params = {**params, "token": token}
        r = requests.get(f"https://{HOST}{path}", headers=headers, params=params, timeout=15)
        r.raise_for_status()
        page = r.json()
        for item in page.get("data", []):
            yield item
        token = page.get("continuation") or page.get("token")
        if not token:
            break
```

## Тарифы и расчёт расходов

> 📌 Этот раздел — для AI-ассистента. Когда пользователь спрашивает "сколько это будет стоить?" — используй данные ниже.

### Тарифные планы (verified из Pricing tab, YYYY-MM-DD)

| План | Запросов/мес | Rate Limit | Цена/мес | Overage |
|------|--------------|------------|----------|---------|
| Basic | _N_ | _N req/час_ | _$0 / $X_ | _hard limit / $X за extra_ |
| Pro | _N_ | _N req/час или нет_ | _$X_ | _hard limit / $X_ |
| Ultra | _N_ | – | _$X_ | _hard limit / $X_ |
| Mega | _N_ | – | _$X_ | _$X за extra-запрос_ |

**Bandwidth (на всех планах):** _N MB/мес включено_ + _$X за каждый дополнительный 1 MB_ (если коннектор отдаёт большие данные; если нет — удали этот блок).

**Features (на всех планах):** _список фичей если они есть_

### Стоимость одного запроса в квоте

| Что делает запрос | Стоимость | Пример |
|---|---|---|
| База (любой запрос) | **1 unit** | `GET /<endpoint>` |
| `+ <модификатор>=1` | **+1 unit** | `/<endpoint>?<modifier>=1` |
| `+ <параметр>` (handle вместо id и т.п.) | **+1 unit** | _когда применяется_ |

### Формула расчёта месячной стоимости

```
quota_per_request = 1 + sum(модификаторов "+1")
month_quota_usage = quota_per_request × запросов_в_месяц
month_bandwidth_mb = средний_размер_ответа_MB × запросов_в_месяц

# Подбор плана — первый, где month_quota_usage ≤ лимит плана
# Bandwidth-overage: (фактические_MB - включённые) × $/MB

total_monthly = plan_price + bandwidth_overage
```
