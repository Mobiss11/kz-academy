# Подключение к Claude.ai (Projects)

В веб-интерфейсе [Claude.ai](https://claude.ai) скиллы как формальная фича могут быть недоступны или ограничены планом. Самый надёжный способ — использовать **Projects**: загрузить файлы в knowledge базу проекта, и Claude будет видеть их во всех чатах этого проекта.

## Шаги

1. Создай новый Project в Claude.ai (нужен Pro или Team план).
2. В разделе **Project knowledge** загрузи как минимум:
   - `SKILL.md`
   - всю папку `connectors/` (по одному файлу)
3. В **Custom instructions** проекта добавь:

   ```
   Этот проект — для работы с RapidAPI. В knowledge есть SKILL.md
   с общими правилами и карточки коннекторов в connectors/. Когда я
   спрашиваю о любом API из RapidAPI — сначала открой SKILL.md, потом
   найди подходящую карточку коннектора и используй её для генерации
   правильного кода.
   ```

4. Сохрани проект.

## Проверка

В чате внутри проекта спроси:

> Напиши Python-скрипт, который ищет видео через yt-api на RapidAPI

Claude должен сослаться на содержимое карточки yt-api.md и использовать правильные хедеры и эндпоинты.

## Ограничения

- Project knowledge — это просто загруженные файлы, без автоматической активации по `description`. Поэтому в custom instructions важно явно сказать "пользуйся этими файлами".
- Объём knowledge ограничен (зависит от плана). Не загружай примеры Python без необходимости — достаточно `.md`-файлов.
- Чтобы обновить — придётся перезагрузить файлы вручную. Можно автоматизировать через [Claude API](https://docs.claude.com/en/api/overview), но это уже отдельная задача.
